 --   Почему PostgreSQL?

  - Free (Open Source)
  - Лучший выбор для изучения: принсталлировал и понеслась!
  -  "Взрослая" СУБД, хорошо поддерживающая транзакционность из коробки
  - Весьма развитый диалект SQL (то есть весьма развиты процедурные расширения)
